GistIt
======

Easily publish text as a Gist on [Github](http://gist.github.com] with a hotkey.

Default hotkey: **Ctrl+Shift+C**

Demo: [@Youtube](https://www.youtube.com/watch?v=k8XVSglbVqM)

##Build Instructions

    $ git clone git@github.com:voldyman/GistIt.git
    $ cd GistIt
    $ ./quickbuild
